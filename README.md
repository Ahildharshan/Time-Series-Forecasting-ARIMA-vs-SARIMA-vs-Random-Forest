# Time-Series-Forecasting-ARIMA-vs-SARIMA-vs-Random-Forest
This project presents a comparative study of three forecasting approaches:  📊 ARIMA (AutoRegressive Integrated Moving Average)  🔁 SARIMA (Seasonal ARIMA)  🌲 Random Forest Regressor  The objective is to evaluate their performance on time series data using statistical diagnostics (ACF &amp; PACF) and machine learning-based forecasting.
